# GraphRAG Frontend

React + TypeScript + Vite client for the GraphRAG backend. See the
[project README](../README.md) for setup, architecture, and usage.

```bash
npm install
npm run dev      # development server on http://localhost:5173
npm run lint     # eslint
npm run build    # typecheck + production build
```

Set `VITE_API_URL` if the backend is not running on `http://localhost:8000`.
